HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat dan menyusun halaman web. HTML bekerja dengan menggunakan berbagai elemen atau tag, seperti h1, p, dan img, untuk menampilkan teks, gambar, tautan, dan elemen lainnya di dalam sebuah halaman. Bisa dibilang, HTML adalah kerangka dasar sebuah website, di mana elemen-elemen ini diatur agar tampilan halaman menjadi terstruktur dan mudah dibaca oleh pengguna. Namun, HTML sendiri tidak mengatur desain atau interaktivitas untuk itu, biasanya digunakan bersama CSS untuk tampilan dan JavaScript untuk fungsi tambahan.
